# Memory Card Matching Game 🎮

An interactive and fun Memory Card Matching Game built using HTML, CSS, and JavaScript.

## 📌 Features

* Random card shuffling
* Card matching logic
* Attempt counter
* Reset game functionality
* Winning message display
* Confetti animation effect
* Simple and responsive UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```bash
Memory-Card-Game/
│
├── index.html
├── style.css
└── script.js
```

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in any browser

## 🎯 Project Description

This project is a browser-based memory matching game where players flip cards to find matching color pairs. The game keeps track of attempts and displays a congratulatory message with confetti animation after all pairs are matched.

## 📸 Game Functionalities

* Flip cards on click
* Match identical cards
* Hide unmatched cards automatically
* Count total attempts
* Restart the game anytime

## 📚 Concepts Used

* DOM Manipulation
* Event Handling
* Arrays and Functions
* CSS Grid Layout
* JavaScript Game Logic
